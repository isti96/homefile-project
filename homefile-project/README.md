1. Open a terminal, navigate to the path where you want to clone the project, clone the repo from GitHub with: git clone https://github.com/isti96/homefile-project.git

2. After cloned, change directory in terminal: cd homefile-project

3. Install dependencies: npm install
Frontend runs with command: ng serve

4. Open another terminal, change directory: cd homefile-project then run backend with command: npm start

5. Open http://localhost:4200/
